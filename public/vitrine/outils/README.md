# Assets pour la page Outils Numériques

Ce dossier doit contenir les assets visuels pour la page des outils numériques (`/vitrine/outils-numeriques`).

## Assets requis

### Background Hero

**hero.png** - Image de fond pour la section Hero
- Format : PNG ou JPG optimisé
- Résolution recommandée : 1440px de largeur minimum
- Utilisé dans : `/vitrine/outils-numeriques` (page.tsx:18)

### Icônes des outils (96x96px)

1. **icon-min.svg** - Icône de Mon inclusion Numérique
   - Format : SVG
   - Couleurs : Bleu (#000091) et violet
   - Icône hexagonale avec logo MIN

2. **icon-coop.svg** - Icône de La Coop de la Médiation
   - Format : SVG
   - Couleurs : Bleu (#000091)
   - Icône en forme de 4 carrés/grille

### Captures d'écran des outils (588x420px environ)

3. **visuel-min.png** - Capture d'écran de Mon inclusion Numérique
   - Doit montrer l'interface de l'outil avec tableaux de bord et statistiques
   - Background suggéré : #F5F5FE

4. **visuel-coop.png** - Capture d'écran de La Coop de la Médiation
   - Doit montrer l'interface avec graphiques et fonctionnalités d'export
   - Background suggéré : #F5F5FE

5. **visuel-cnfs.png** - Capture d'écran du Tableau de pilotage Conseiller Numérique
   - Doit montrer l'interface de pilotage
   - Background suggéré : #F5F5FE

## Notes

- Tous les visuels doivent être optimisés pour le web
- Les captures d'écran doivent être récentes et représentatives des outils
- Les icônes SVG doivent être nettoyées et optimisées
- Vous pouvez extraire ces assets depuis le design Figma : https://www.figma.com/design/IFH80doDOEvJvbMlUnQIOC/%E2%AD%90%EF%B8%8F-Outil-gestionnaire-FNE?node-id=10525-15843
